# Temperature-Monitoring-System

The Temperature Monitoring System is an Arduino-based project designed to measure and display real-time ambient temperature using a temperature sensor (e.g., LM35 or DHT11) and a 16x2 LCD display. 
The sensor provides analog or digital data to the Arduino, which processes the input and continuously updates the temperature reading on the LCD in degrees Celsius.
This system can be used in applications such as environmental monitoring, greenhouse control, HVAC systems, or educational projects to demonstrate the use of sensors, analog/digital input, and real-time display in embedded systems.

# Components Used:-
Arduino, Temperature Sensor, Breadboard, LCD Display, Jumper wires.
